## üìä Research Methodology

**Phase 1: Primary Source Collection**
- Web searches for official announcements, press releases, blog posts
- ICP Forum deep dive for project/hub reports
- Dashboard data (IC Dashboard, on-chain metrics)
- Archive.org backups for all sources

**Phase 2: Verification**
- Every quantitative claim gets a source URL
- Confidence level assigned (HIGH/MEDIUM/LOW)
- If no source found after thorough search: Label "UNVERIFIED" or exclude

**Phase 3: Conservative Analysis**
- Only analyze data with HIGH/MEDIUM confidence
- Extrapolations clearly labeled with methodology disclosed
- Explicit "What We Don't Know" sections
