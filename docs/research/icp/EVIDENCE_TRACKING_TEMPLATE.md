## RESEARCH PROTOCOL

### Phase 1: Primary Source Collection (2-3 days)

**Day 1-2: Web Research**
- ICP Forum deep dive (all project announcements, hub reports, event recaps)
- Twitter/X archive (DFINITY, projects, hubs, founders)
- Project blogs (Medium, official sites)
- Dashboards (IC Dashboard, IC Rocks, OpenChat analytics if public)
- Devpost (hackathons)
- YouTube (event recordings, founder interviews)

**Day 3: Data Entry**
- Log every found data point in this template
- Create archive.org backups
- Assign confidence levels
- Flag gaps requiring follow-up research

### Phase 2: Verification & Triangulation (1-2 days)

**Day 4-5: Cross-Verification**
- For critical claims: Find 2-3 independent sources
- For conflicting data: Document discrepancies
- For unsourced claims: Label as "UNVERIFIED - needs primary source"

### Phase 3: Analysis (1 day)

**Day 6: Pattern Analysis**
- Only analyze data with HIGH/MEDIUM confidence
- Calculate CAC ranges (show math, disclose assumptions)
- Identify what works vs what doesn't (based on verified data only)
- Acknowledge knowledge gaps explicitly

### Phase 4: Report Creation (1 day)

**Day 7: Write Verified Report**
- Every claim cites this evidence database
- Every URL is clickable and archived
- Extrapolations clearly labeled
- Conservative estimates only
- "What We Don't Know" section
